# PWA Icon Generator

You need to create two icon files for the PWA to work properly:

1. **icon-192x192.png** - 192x192 pixels
2. **icon-512x512.png** - 512x512 pixels

## Quick Options:

### Option 1: Use an Online Tool
- Visit https://realfavicongenerator.net/ or https://www.pwabuilder.com/imageGenerator
- Upload your logo/icon
- Download the generated icons
- Place them in the `/public` folder

### Option 2: Create Simple Icons
You can create simple colored squares with text using any image editor:
- 192x192px and 512x512px PNG files
- Use your brand color (#f97316 - orange)
- Add "HT" or your logo in the center
- Save as `icon-192x192.png` and `icon-512x512.png` in `/public`

### Option 3: Use a Placeholder
For testing, you can use any 192x192 and 512x512 PNG images temporarily.

## Icon Requirements:
- Format: PNG
- Sizes: 192x192 and 512x512 pixels
- Purpose: Should be recognizable at small sizes
- Background: Can be transparent or solid color
